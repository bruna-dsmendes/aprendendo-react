# 📚 Aprendendo React

Repositório de estudos e experimentos com **React**, desenvolvido como um playground pessoal para praticar conceitos fundamentais e avançados da biblioteca.

---

## 🚀 Tecnologias

- [React 18](https://react.dev/) — biblioteca para construção de interfaces
- [TypeScript](https://www.typescriptlang.org/) — tipagem estática para JavaScript
- [Vite](https://vitejs.dev/) — bundler rápido com Hot Module Replacement (HMR)
- [React Router DOM v7](https://reactrouter.com/) — roteamento client-side
- [ESLint](https://eslint.org/) — linting e padronização de código

---

## ⚙️ Como rodar o projeto

### Pré-requisitos

- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- npm ou yarn

### Instalação

```bash
# Clone o repositório
git clone https://github.com/bruna-dsmendes/aprendendo-react.git

# Acesse a pasta do projeto
cd aprendendo-react

# Instale as dependências
npm install
```

### Scripts disponíveis

```bash
# Inicia o servidor de desenvolvimento
npm run dev

# Gera o build de produção
npm run build

# Visualiza o build de produção localmente
npm run preview

# Executa o linter
npm run lint
```
